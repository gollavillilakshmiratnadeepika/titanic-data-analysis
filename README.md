# Titanic Survival Analysis

This project analyzes the Kaggle Titanic dataset using Python and Pandas.

## What I did
- Cleaned missing values (Age, Embarked) and removed unusable columns (Cabin)
- Explored survival trends by gender, passenger class, and age
- Learned data cleaning and exploratory data analysis using Python and Pandas

## Tools used
- Python
- Pandas
- Kaggle Notebook
